# Como colocar a loja no GitHub

## Opcao sem instalar Git

1. Acesse https://github.com/new
2. Nome do repositorio sugerido: `johnvisionseg-loja`
3. Marque como `Public`.
4. Clique em `Create repository`.
5. Clique em `uploading an existing file`.
6. Arraste todos os arquivos da pasta `JohnVisionSeg_Loja_Online_Produtos`.
7. Clique em `Commit changes`.
8. Va em `Settings > Pages`.
9. Selecione `Deploy from a branch`, branch `main`, pasta `/root`.
10. Aguarde o link ficar pronto.

## Nome do site gratis

O GitHub vai gerar algo como:

`https://SEU-USUARIO.github.io/johnvisionseg-loja/`

## Importante

Se a loja carregar uma versao antiga, use `Ctrl + F5` no navegador.
